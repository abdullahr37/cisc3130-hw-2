Student Management System
=========================

Description
-----------
In this project, I created a Student Management System using ArrayLists and Generics in Java.

The program allows me to:
- Add and remove students
- Add courses
- Enroll students in courses
- Assign grades
- Calculate GPA
- Generate simple reports
- Demonstrate ArrayList operations
- Use generic classes like Stack, Queue, Pair, and utility methods

How to Compile
--------------
Open the folder in the terminal and run:

javac *.java

How to Run
----------
To run the main program:

java StudentManagementSystemMain

Design Decisions
----------------
I used ArrayList instead of arrays because it resizes automatically and makes adding and removing elements easier.  
I used student ID and course code as unique identifiers.  
GPA is calculated based on assigned grades in enrollments.

What I Learned
--------------
From this project, I learned:
- How ArrayList works and how it resizes
- The difference between Array and ArrayList
- How to write generic methods and classes
- How bounded generics and wildcards work
- How to build Stack and Queue using ArrayList

Array vs ArrayList
------------------
Arrays have a fixed size and cannot grow.  
ArrayLists can grow and shrink dynamically.  

I found ArrayList much easier to use for this type of application.
